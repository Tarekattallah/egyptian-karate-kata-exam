# امتحان الكاتا - الاتحاد المصري للكاراتيه

تطبيق ويب لامتحان الكاتا (Kata) مخصص للاتحاد المصري للكاراتيه.

## التقنيات

- HTML5
- CSS3
- Vanilla JavaScript
- JSON

## هيكل المشروع

```
exam-ui/
├── index.html
├── css/
│   ├── style.css
│   ├── animations.css
│   └── responsive.css
├── js/
│   ├── app.js
│   └── timer.js
└── data/
    └── questions.json
```

## الميزات

- عرض سؤال واحد في كل مرة
- لوحة أرقام للوصول السريع لجميع الأسئلة
- أزرار تنقل (السابق / التالي)
- مؤقت 60 دقيقة
- شريط تقدم
- إحصائيات (صحيحة / خاطئة / لم تتم)
- فيدباك فوري لكل إجابة
- شاشة نتائج نهائية
- زر إعادة الامتحان
- فوتر مع اسم المطور

## التشغيل

افتح `index.html` في المتصفح.

## البيانات

الأسئلة موجودة في `data/questions.json`.

## التصميم

- متجاوب مع جميع أحجام الشاشات
- يدعم اللغة العربية (RTL)
- تصميم بسيط ورسمي
